cm14-skel
=========

Forkable Composer skeleton project for Codemotion 2014

 * Fork it on github
 * create and move to your working directory
   * `mkdir <dirname>`
 * configure upstream & pull it
   * `git init` 
   * `git remote add origin <your-fork>`
   * `git fetch`
   * `git pull origin master`
 * Install vendors
   * `composer update`
 *  Run tests
   * `php bin/phpunit`
